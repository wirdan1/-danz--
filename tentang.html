Tentang Website Ini

Website ini dibuat sebagai portofolio pribadi untuk menunjukkan keahlian dalam pengembangan web dan teknologi modern.

Teknologi yang Digunakan

Teknologi Deskripsi
HTML5 Struktur dasar halaman web yang modern
CSS3 Menggunakan animasi, grid, dan flexbox
JavaScript Menambahkan interaksi dan animasi pada website
Chart.js Visualisasi data menggunakan grafik interaktif

Visi & Misi

Website ini bertujuan untuk:

  • Meningkatkan kesadaran tentang pentingnya pengembangan web yang orisinal.
  • Membantu pengembang baru dalam memahami teknologi modern.
  • Menyediakan proyek open-source untuk referensi belajar.
